# Threads Auto Expand Spoilers

Automatically expands all spoilers on Threads / Facebook.

## Features
- Auto expand text spoilers
- Auto expand image/video spoilers
- MutationObserver (real-time)
- Optional auto scroll
- Tampermonkey menu controls
